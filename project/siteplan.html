Site Plan: Kell's Top Favorite Games

Site Name

I chose this name because it represents the purpose of the site accurately.

Site Purpose

The site will show Kell Brine's top 10 favorite games and game releases he's looking forward to. It will also provide a way to contact him.

Scenarios

  • Where can I find contact information for Kell Brine?
